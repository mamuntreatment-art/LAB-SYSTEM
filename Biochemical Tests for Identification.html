Biochemical Tests for Identification

Biochemical tests ব্যবহার করে bacteria এর metabolic activity detect করা হয়। এই test গুলো organism identification এর জন্য অত্যন্ত গুরুত্বপূর্ণ।

Common Biochemical Tests

Test Name Purpose Positive Result Example Organism
Gram Stain Gram positive / negative differentiation Purple or Pink cells All bacteria
Catalase Test Hydrogen peroxide breakdown Bubble formation Staphylococcus spp.
Coagulase Test Detect clotting enzyme Clot formation Staphylococcus aureus
Oxidase Test Cytochrome oxidase enzyme detection Dark purple color Pseudomonas aeruginosa
Indole Test Tryptophan breakdown detection Red ring formation Escherichia coli
Urease Test Urea hydrolysis detection Pink color Proteus spp.
Citrate Test Citrate utilization detection Blue color Klebsiella spp.
TSI Test Sugar fermentation & H₂S production Yellow / Black butt Salmonella spp.
Motility Test Bacterial movement detection Diffuse growth Proteus spp.
Bile Esculin Test Esculin hydrolysis detection Blackening of media Enterococcus spp.
Optochin Test S. pneumoniae identification Zone of inhibition Streptococcus pneumoniae
Bacitracin Test S. pyogenes identification Sensitive zone Streptococcus pyogenes
Germ Tube Test Candida albicans detection Germ tube formation Candida albicans
